# Gestion Hotel

Système de gestion des réservations d’hôtels.

## Technologies utilisées
- Laravel
- MySQL
- Bootstrap
- JavaScript

## Fonctionnalités
- Authentification des utilisateurs
- Gestion des chambres
- Réservations
- Génération de PDF
- Envoi d’emails de confirmation
